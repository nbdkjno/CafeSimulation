# CafeSimulation
CafeSimulator — это 2D-игра в жанре симулятора кафе, разработанная на игровом движке Unity с использованием языка C#. Игрок выступает в роли бариста, задача которого — правильно и аккуратно обслуживать посетителей, собирая заказы методом перетягивания.

Ключевые особенности:

- Три уровня сложности (с каждым уровнем время выполнения заказа увеличивается, а количество позиций в ступени увеличивается).
- Система жизни (три жизни, потеря денег или запас времени).
- Счётчик успешных заказов (три успешных заказа повышают уровень).
- На экране «GAME OVER» можно начать или снова вернуться в меню.
- Пиксельная графика.
- Фоновая музыка и звуковые эффекты с отключением.
- Управление: перетаскивание предметов мышью.

Использованные технологии Разработка игры «CafeSimulator» осуществлялась с использованием игрового движка Unity 2022 LTS. Языком программирования выбран C#. Анимации персонажей и объектов создаются через систему Animator. Звуковое сопровождение, включающее фоновую музыку и звуковые эффекты, реализованное с использованием компонента AudioSource. Для контроля управления и хостинга проекта были использованы Git и платформа GitHub.

Инструкция по запуску Для запуска готовой сборки игры необходимо перейти в раздел Репозиторий релизов на GitHub, скачать полную версию архива и распаковать его в любой адрес на компьютере. После этого следует запустить исполняемый файл CafeSimulator.exe. Для запуска проекта в режиме разработчика необходимо обратиться к репозиториям с помощью команды git clone, откройте проект в Unity Editor версии 2022 LTS или выше и нажмите кнопку Play в редакторе. Если игра не запускается, необходимо убедиться, что на компьютере установлены соответствующие компоненты .NET Framework и распространяемый компонент Visual C++, которые обычно уже присутствуют в операционных компонентах Windows 10 и 11.
